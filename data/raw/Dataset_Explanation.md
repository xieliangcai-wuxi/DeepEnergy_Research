
# 多模态短期电力负荷预测数据集字段说明
## 一、 负荷及发电量相关数据集字段
| 列名 | 字段含义 | 备注 & 用法 |
|-----|---------|------------|
| time | 时间戳 | 用于对齐天气数据 |
| generation ... | 各种发电量 | 历史数据，预测未来时未知；仅能作为**过去24小时的滞后特征（Lag features）** 输入；禁止将 t 时刻发电量作为 t 时刻负荷的输入 |
| forecast solar/wind | 风/光发电预测 | 强特征；电网运营商（TSO）提前发布的次日新能源预测；预测次日负荷时为合法输入特征 |
| total load forecast | 官方负荷预测 | **绝对不能作为模型输入**；若作为输入，模型仅会模仿现有系统而非自主预测；仅用于**基准对比（Benchmark）**，验证自研模型的优越性 |
| total load actual | 实际总负荷 | 预测目标/标签（Target，即模型的 y 值） |
| price ... | 电价 | 数值特征 |

## 二、 气象相关数据集字段
| 列名 | 字段含义 | 备注 & 用法 |
|-----|---------|------------|
| dt_iso | 时间戳 | 数据集主键；用于与负荷表进行数据对齐 |
| city_name | 城市名 | 空间索引；西班牙5个不同方位城市，代表全国气象分布特征 |
| temp, min, max | 温度（含最高/最低温） | 数值特征；单位为开尔文（Kelvin），**建议转换为摄氏度**；是与负荷相关性最强的变量 |
| pressure, humidity | 气压、湿度 | 数值特征；影响体感温度，进而影响空调类负荷需求 |
| wind_speed, deg | 风速、风向 | 数值特征；影响风力发电量（关联净负荷），同时影响环境散热效率 |
| rain_1h...clouds | 降雨量、云量 | 数值特征；云量直接影响光照强度（关联太阳能发电量）和照明负荷需求 |
| weather_main/desc | 天气语义描述 | 文本/语义模态特征；示例：`sky is clear`、`light rain` |
| weather_icon/id | 天气图标/标识ID | 冗余信息，无实际建模价值 |
